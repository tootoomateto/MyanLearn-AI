# 🌟 MyanLearn-AI

> Empowering Myanmar's Youth with AI-Driven Personalized Learning.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Community: Discord](https://img.shields.io/badge/Community-Discord-blue)](https://discord.gg/xsdwPgR9Y)

**MyanLearn AI** ဆိုသည်မှာ မြန်မာနိုင်ငံ၏ ရိုးရာအလွတ်ကျက်စနစ်ကို တော်လှန်ပြောင်းလဲရန်နှင့် ကျောင်းသားတစ်ဦးချင်းစီ၏ ထူးခြားသော အရည်အချင်းများကို AI နည်းပညာဖြင့် ဖော်ထုတ်ပေးရန် ရည်ရွယ်သည့် Open-source ပရောဂျက်ဖြစ်သည်။

---

## 🚀 Vision (မျှော်မှန်းချက်)
ကျွန်ုပ်တို့၏ ရည်မှန်းချက်မှာ ဘာသာစကား အခက်အခဲနှင့် သင်ကြားမှုစနစ် အားနည်းချက်များကြောင့် နောက်ကျကျန်ခဲ့သော မြန်မာလူငယ်များကို ကမ္ဘာ့အဆင့်မီ ဗဟုသုတများနှင့် ချိတ်ဆက်ပေးရန်ဖြစ်သည်။

## ✨ Key Features (အဓိက ပါဝင်မည့်အရာများ)
* **AI Personal Tutor:** ကျောင်းသား၏ လေ့လာမှုနှုန်းအပေါ် မူတည်၍ ပုံစံပြောင်းလဲ သင်ကြားပေးမည့် AI။
* **Knowledge Base:** ကမ္ဘာပေါ်ရှိ နည်းပညာနှင့် ဗဟုသုတများကို မြန်မာလို လွယ်ကူစွာ လေ့လာနိုင်ခြင်း။
* **Project Collaboration:** လူငယ်ချင်း ချိတ်ဆက်၍ လက်တွေ့ Project များ အတူတူ ဖန်တီးနိုင်ခြင်း။
* **Skill Portfolio:** အမှတ်စာရင်းထက် လက်တွေ့အရည်အချင်းကို ဖော်ပြပေးမည့် Portfolio စနစ်။

## 📅 Roadmap (လုပ်ငန်းစဉ်များ)

### 🟢 Phase 1: The Seed (Current)
- [ ] AI Tutor Engine (RAG implementation) တည်ဆောက်ခြင်း။
- [ ] အခြေခံ Web UI (Dashboard) ဖန်တီးခြင်း။
- [ ] DeepSeek API Integration လုပ်ခြင်း။

### 🟡 Phase 2: The Foundation
- [ ] User Authentication & Profiles။
- [ ] Shared Project Workspaces။
- [ ] Peer-to-peer Messaging စနစ်။

### 🔴 Phase 3: The Growth
- [ ] Mobile Application (Android/iOS) ထုတ်လုပ်ခြင်း။
- [ ] Offline Learning Mode ထည့်သွင်းခြင်း။

## 🤝 How to Contribute (ပါဝင်ကူညီရန်)
ကျွန်ုပ်တို့သည် အောက်ပါ Role များအတွက် ဝါသနာရှင်များကို ဖိတ်ခေါ်နေပါသည်-
1.  **Developers:** Python (FastAPI), React, AI/ML Engineers။
2.  **UI/UX Designers:** ကျောင်းသားများအတွက် သုံးရလွယ်ကူသော Interface ဖန်တီးရန်။
3.  **Educators:** သင်ရိုးညွှန်းတမ်းများကို စစ်ဆေးရန်နှင့် လမ်းညွှန်ရန်။

ပါဝင်လိုပါက [Contribution Guide](CONTRIBUTING.md) ကို ဖတ်ရှုပေးပါ။
